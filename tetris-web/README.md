# Tetris Web

シンプルなブラウザ版テトリスです。

使い方:

- 開発中は `index.html` をブラウザで直接開くか、シンプルなHTTPサーバで配信してください。

ローカルでの実行例:

```bash
# Python 3 の簡易サーバ
python3 -m http.server 8000
# ブラウザで http://localhost:8000/tetris-web/ を開く
```

操作方法:

- 矢印キー: 左右移動/ソフトドロップ
- 上キー / Z: 回転
- Space: ハードドロップ
- P: ポーズ
- 画面のボタンでも操作可能
